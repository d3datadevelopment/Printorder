### Backend

* ID des verwendeten Themes eintragen:

    [ Erweiterungen ] -> [ Module ] -> [ {$menutitle} ] -> [ Tab: Einstell. ] -> [ Option: Template-Blöcke dem eigenen Theme zuordnen (Einstellung nur für dieses Modul) ]

* Pfad für die Ablage der Bestellbestätigungen hinterlegen:

    [ D3 Module ] -> [ {$menutitle} ] -> [ Einstellungen ] -> [ Stamm]
